# Typescript: Compare Article Content Using Diff Utility

**Installation**

```sh
npm install

# or yarn

yarn
```


**Random Comparison**

```
npm run start
```

**Run Diff Comparison by Character**

```
npm run chardiff

# or by specific file

npm run chardiff dataset/1mdb-en-1.txt 
```

**Run Diff Comparison by Word**

```
npm run worddiff

# or by specific file

npm run worddiff dataset/1mdb-en-1.txt 
```
